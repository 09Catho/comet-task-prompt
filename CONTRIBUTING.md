## Contributing

1. Copy `prompts/misc/template.md` into the right folder.
2. Fill the YAML header using fields in `schema.yaml`.
3. Use kebab-case for filenames.
4. Keep prompts explicit, step-by-step, with clear stop conditions.
5. Use placeholders like `{{LANG}}`, `{{DATE}}`, `{{TOPIC}}`.

Open a PR when ready.
